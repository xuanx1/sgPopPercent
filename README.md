# Singapore's Projected Age Distribution [Preview](https://xuanx1.github.io/sgPopPercent/singapore_age_distribution.html)

## 📊 Project Overview

This project analyzes and visualizes Singapore's demographic transition using UN World Population Prospects (WPP) 2024 data, showing how the population's age structure evolves from 1950 to 2100. The interactive visualization demonstrates Singapore's transformation from a young, growing population to an aging society over 150 years.

## 🎯 Key Features

- **Interactive Timeline**: Explore demographic changes across 150 years (1950-2100)
- **Age Group Analysis**: Detailed breakdown of Children (0-14), Working Age (15-64), and Elderly (65+) populations
- **Smooth Animations**: Watch Singapore's demographic transition unfold in real-time
- **Comprehensive Data**: Single-year age groups (0-100+) with 50+ demographic indicators
- **Professional Visualization**: Clean, responsive design optimized for all devices

## 📈 Data Sources & Methodology

- **Primary Data**: UN World Population Prospects 2024
- **Historical Period**: 1950-2023 (Estimates)
- **Projection Period**: 2024-2100 (Medium Variant)
- **Geographic Focus**: Singapore
- **Age Resolution**: Single-year age groups (0-100+)

## 📊 Key Demographic Insights

The visualization reveals Singapore's dramatic demographic transition:

- **1950s**: Young population with high birth rates
- **2023**: Mature population with below-replacement fertility
- **2050+**: Rapidly aging society with inverted age pyramid
- **2100**: Projected elderly-dominant population structure
